# MÔN ĐỒ ÁN CHUYÊN NGÀNH - PHÂN TÍCH VÀ MÔ HÌNH HÓA THUẬT TOÁN

Dự án này là đóng góp cá nhân của tôi trong khuôn khổ môn **Đồ án Chuyên ngành**, tập trung vào việc hiểu sâu sắc kiến thức nền tảng về thuật toán Machine Learning.

---

## Đóng góp Cá nhân & Vai trò Lãnh đạo (Team Leader)

Repository này **chỉ chứa các file code và tài liệu** do cá nhân tôi thực hiện trong dự án. Do đó, Repository này có thể **không chạy được hệ thống hoàn chỉnh** mà chỉ dùng để chứng minh năng lực kỹ thuật nền tảng.

* **Vai trò:** **Team Leader** (Dẫn dắt nhóm 4 người)
* **Trách nhiệm Lãnh đạo:** Dẫn dắt nhóm triển khai dự án, chịu trách nhiệm quản lý phiên bản code và đảm bảo các mốc thời gian kỹ thuật.

---

## Kỹ thuật Cốt lõi (Core Technical Contribution)

Phần việc này tập trung vào việc xây dựng nền tảng thuật toán và so sánh hiệu suất:

1.  **Tiền xử lý Dữ liệu:** Thực hiện **tiền xử lý dữ liệu** chuyên sâu (Data Pre-processing) để chuẩn hóa và làm sạch tập dữ liệu đầu vào.
2.  **Mô hình hóa Thủ công (From Scratch):** **Trực tiếp code thủ công (from scratch)** các thuật toán Machine Learning cơ bản (không dùng thư viện Scikit-learn):
    * **Linear Regression** (Hồi quy Tuyến tính)
    * **Logistic Regression** (Hồi quy Logistic)
3.  **Phân tích Hiệu suất:** Thực hiện **so sánh hiệu suất trực quan** giữa mô hình được code thủ công và mô hình tương đương được xây dựng bằng thư viện **Scikit-learn**, nhằm chứng minh sự hiểu biết vững chắc về cách thức hoạt động của thuật toán.
